# Documentación (`anonuevo/docs/`)

Este directorio agrupa la documentación del sistema `anonuevo` (motor de hojas `/vivos/*`, panel `control.html`, y servidor `control-server.js`).

## Índice rápido

- `APIS_DISPONIBLES.md`: endpoints y servicios expuestos por el server.
- `AI_SERVICES.md` / `IA_USO_DETALLADO.md`: uso de IA y proveedores.
- `CIUDAD_EN_VIVO.md`: endpoints/proxies “city live”.
- `URLS_HOJAS_DINAMICAS.md` / `PROPUESTA_NUEVAS_HOJAS.md` / `NUEVAS_APIS_PROPUESTAS.md`: roadmap de hojas.
- `INSTALACION_TTS.md` / `INSTALACION_EDGE_TTS.md` / `QUICK_START_EDGE_TTS.md` / `VOZ_CLONACION_OPCIONES.md`: TTS.
- `AUDITORIA_LIBRO.md`: auditoría/estructura del “libro”.
- `VIDEO_LOOPS.md`: loops/recursos para transmisión.
- `RESUMEN_EJECUTIVO.md`: resumen alto nivel.
- `_archive/studio/`: documentación histórica de “Studio” (removido del runtime).
