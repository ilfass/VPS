# Instrucciones para agregar la foto del presentador

Para que el presentador muestre tu cara, necesitas agregar una imagen llamada `presenter-face.jpg` en la carpeta `anonuevo/`.

## Pasos:

1. Prepara una foto tuya (preferiblemente de la cara, tipo selfie)
2. Renómbrala como `presenter-face.jpg`
3. Colócala en la carpeta `/home/fabian/Documentos/VPN/VPS/anonuevo/`
4. La imagen debe ser cuadrada o casi cuadrada para verse bien (recomendado: 400x400px o más)
5. El sistema recortará automáticamente la imagen para que se ajuste al avatar circular

## Formato recomendado:
- **Formato**: JPG o PNG
- **Tamaño**: 400x400px o más
- **Orientación**: Vertical u horizontal (se ajustará automáticamente)
- **Calidad**: Buena resolución para que se vea nítida

## Nota:
Si no agregas la imagen, el avatar mostrará un fondo oscuro. Una vez que agregues `presenter-face.jpg`, el avatar se actualizará automáticamente.
